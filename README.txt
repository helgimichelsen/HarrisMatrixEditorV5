Harris Matrix Editor 5.0 HMC

Videreudvikling af V4 inspireret af Harris Matrix Composer-artiklen.

Nyt i V5:
- HMC-lignende minimal startmatrix: Top surface, Unexcavated og Interface to geology.
- Unit-typer: Deposit og Surface/Interface.
- Stratigrafiske relationer: source above target.
- Temporal relations: later og contemporary.
- Dotted/purple temporal lines.
- Phase og Period grupper, med collapse/expand.
- Properties panel.
- Relations editor panel.
- Search.
- Bookmarks.
- Zoom med mouse wheel + zoom knapper.
- Validity check:
  * cycles
  * missing above/below relationer
  * invalid Top/Geology relationer
- Auto-layout:
  * stratigraphic/later vertical top-to-bottom
  * contemporary same level
  * transitive reduction
- Import/export HMCX (generic XML/HMCX).
- Export SVG/PDF.
